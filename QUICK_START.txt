```css

```

### Change Library Name


### Add More Books

```javascript
AppState.books.push({
  id: 7,
  title: 'New Book',
  // ... rest of data
});
```

### Change Theme Default
Edit `app.js` line 7:
```javascript
isDarkTheme: false, // true for dark, false for light
```

## üêû Troubleshooting

### Page Won't Load
- Make sure all files are in same folder:
  - `bygithub.html`
  - `styles.css`
  - `app.js`

### Styles Look Wrong
- Verify `styles.css` is in same folder
- Clear browser cache (Ctrl+Shift+Del)
- Refresh page (Ctrl+R)

### JavaScript Errors
- Open DevTools (F12)
- Check Console tab for errors
- Make sure `app.js` is linked in HTML

### Can't Login
- Any email/password works in demo mode
- Try: `test@test.com` / `test`
- Check browser console for errors

### Mobile Layout Broken
- Ensure viewport meta tag is present
- Zoom out browser to test mobile view
- Check responsive at 375px, 768px, 1024px
