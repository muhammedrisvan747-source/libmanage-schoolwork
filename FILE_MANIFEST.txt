TROUBLESHOOTING
───────────────────────────────────────────────────────────────────────────────

PROBLEM: Page won't load
  Solution: Make sure all 3 files are in same folder

PROBLEM: Styles look wrong / missing
  Solution: 
    • Refresh browser (F5 or Ctrl+R)
    • Clear cache (Ctrl+Shift+Del)
    • Verify styles.css exists in same folder

PROBLEM: Features not working
  Solution:
    • Check JavaScript console (F12)
    • Verify app.js is in same folder
    • Refresh page

PROBLEM: Login doesn't work
  Solution:
    • Try any email/password (demo mode)
    • Check browser console for errors

PROBLEM: Mobile layout broken
  Solution:
    • Zoom out to test mobile view
    • Resize browser window
    • Check viewport meta tag in HTML
